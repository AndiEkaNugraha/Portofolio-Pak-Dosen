ğŸ“¦ Installation

Requirements:

  • Odoo 16.0 or higher
  • Python 3.8+
  • Dependencies: website, website_blog, mail, dan modul portfolio lainnya

Installation Steps:

  1. Copy folder profile_dosen ke direktori addons
  2. Restart Odoo server
  3. Update Apps List
  4. Install "Portofolio Dosen - Profile Lengkap"

âš™ï¸ Configuration

Setup Profile:

  1. Buka menu Profile Dosen > Profile > Profile Dosen
  2. Create new profile atau edit existing
  3. Isi semua informasi yang diperlukan
  4. Tandai "Tampilkan di Website" untuk publish

Configure CV Template:

  1. Buka menu Profile Dosen > CV Management > Template CV
  2. Edit template atau create new custom template
  3. Atur sections yang akan disertakan
  4. Set sebagai default template jika diinginkan
