âš¡ Instalasi

Langkah Instalasi:

  1. Copy folder grup_riset ke direktori addons Odoo
  2. Restart server Odoo
  3. Aktifkan mode developer di Odoo
  4. Update App List
  5. Install modul "Grup Riset & Laboratorium"
  6. Konfigurasi website dan menu navigasi
# Update addons list ./odoo-bin -u grup_riset -d your_database

ğŸš€ Penggunaan

Setelah instalasi, Anda dapat:

  • Mengelola kategori grup riset melalui menu "Grup Riset > Kategori"
  • Menambah grup riset baru melalui menu "Grup Riset > Grup Riset"
  • Mengupload foto tim dan laboratorium
  • Mengatur informasi kontak dan fasilitas
  • Mempublikasikan ke website dengan URL: /grup-riset
  • Mengoptimalkan SEO dengan meta tags dan slug URL
