# Portofolio Dosen - Acara Dosen

Modul ini digunakan untuk mengelola daftar acara yang diisi oleh dosen dalam portofolio akademik.

## Fitur Utama

- **Manajemen Acara**: Kelola berbagai jenis acara yang diisi dosen
- **Kategori Acara**: Organisir acara berdasarkan kategori dan jenis
- **Integrasi Website**: Tampilkan acara di website portofolio menggunakan sistem blog Odoo
- **Peran Dosen**: Catat peran dosen dalam acara (pembicara, juri, moderator, dll.)
- **Format Acara**: Mendukung acara offline, online, dan hybrid
- **Dokumentasi**: Simpan sertifikat dan link dokumentasi acara

## Jenis Acara

- Penyuluhan
- Webinar
- Workshop
- Juri/Penilai
- Seminar
- Konferensi

## Peran Dosen

- Pembicara/Penyuluh
- Moderator
- Juri/Penilai
- Penyelenggara
- Peserta
- Lainnya

## Instalasi

1. Copy folder `acara_dosen` ke direktori addons Odoo
2. Update list aplikasi di Odoo
3. Install modul "Portofolio Dosen - Daftar Acara Dosen"

## Penggunaan

1. Akses menu "Acara Dosen" > "Daftar Acara"
2. Buat kategori acara terlebih dahulu
3. Tambahkan acara dengan detail lengkap
4. Publikasikan acara untuk ditampilkan di website
