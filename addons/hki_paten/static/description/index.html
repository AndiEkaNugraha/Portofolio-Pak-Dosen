🏛️ HKI & Paten Manager

Kelola Hak Kekayaan Intelektual & Paten dengan Mudah

Plugin komprehensif untuk mengelola dan mempublikasikan daftar HKI & Paten dalam website portofolio dosen.

📋 Fitur Utama

  • Manajemen HKI Lengkap - Kelola Hak Cipta, Paten, Merek, Desain Industri
  • Kategori Fleksibel - Organisir berdasar jenis, bidang keilmuan, kantor pendaftaran
  • Website Terintegrasi - Publikasi otomatis di website dengan URL /hki
  • SEO Optimized - Meta tags, friendly URL, structured data
  • Responsive Design - Tampilan optimal di desktop, tablet, dan mobile
  • Filter & Search - Pencarian canggih dengan multiple filter
  • Status Tracking - Lacak status dari permohonan hingga pendaftaran
  • Document Management - Upload sertifikat dan dokumen HKI

🎯 Jenis HKI yang Didukung

Hak Cipta Paten Paten Sederhana Merek Desain Industri Desain Tata Letak Varietas Tanaman Rahasia Dagang

🌐 Kantor Pendaftaran

Kemenkumham RI USPTO EPO JPO KIPO SIPO WIPO

📊 Status Tracking

  • Draft - Tahap persiapan dokumen
  • Diajukan - Sudah diajukan ke kantor pendaftaran
  • Dalam Pemeriksaan - Proses evaluasi oleh examiner
  • Dipublikasi - Telah dipublikasikan untuk masa oposisi
  • Diberikan/Terdaftar - HKI telah resmi terdaftar

🔍 Fitur Website

  • Halaman Utama - /hki dengan grid view dan filtering
  • Detail HKI - Informasi lengkap dengan SEO-friendly URL
  • Kategori - Browsing berdasar kategori HKI
  • Pencarian Lanjutan - Multi-criteria search
  • Statistik - Dashboard visual dengan charts

⚙️ Instalasi

  1. Copy folder hki_paten ke direktori addons/
  2. Update Apps List di Odoo
  3. Install module "Daftar HKI & Paten"
  4. Akses menu HKI & Paten di backend
  5. Kunjungi /hki di website untuk melihat publikasi

🎨 Kustomisasi

  • Template Customizable - Sesuaikan tampilan dengan branding
  • Field Extensions - Tambah field custom sesuai kebutuhan
  • Multiple Categories - Buat kategori berdasar kebutuhan
  • Workflow States - Sesuaikan status dan workflow

📱 Kompatibilitas

  • Odoo 19 - Fully compatible dengan Odoo 19
  • Website Module - Requires website, website_blog
  • Responsive - Bootstrap 4+ compatible
  • Modern Browsers - Chrome, Firefox, Safari, Edge
