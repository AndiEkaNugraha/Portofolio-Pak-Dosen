## Export/Import Profil

Untuk backup/restore profil:

1. **Export**: Pergi ke profil list → Select → Actions → Export
2. **Import**: Pergi ke profil list → Actions → Import

Format file yang didukung: CSV, XLS, XLSX

---

**Notes:**
- Semua field HTML dapat diedit dengan rich text editor
- Foto akan auto-resize untuk performa optimal
- Slug tidak boleh duplikat
- CV file bisa dalam format PDF, DOC, DOCX, XLS, etc.
